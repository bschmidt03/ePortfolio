# ePortfolio

## Self Assessment
Throughout this program, my coursework and ePortfolio helped me understand my strengths, improve my skills, and refine my career direction. I originally wanted to go straight into full stack development, then considered help desk as a more realistic starting point, but through building these projects I realized I still want to push toward development long term. This portfolio reflects that by showing both what I’ve built and how I’ve improved it.
I’ve gained experience with collaboration through version control and structured project work, and I’ve improved how I communicate technical ideas through documentation and explanations. Technically, I’ve developed skills in full stack development, algorithms, databases, and security. I’ve worked with frontend and backend systems, improved logic and efficiency in programs, designed database features, and implemented authentication and validation to make applications more secure.
The artifacts in my ePortfolio represent growth across these areas. My Weight Tracker shows software design and usability improvements, my Treasure Hunter project demonstrates algorithmic problem solving, and my Travlr application shows full stack and database integration. Together, they show my ability to build, improve, and explain real applications, and they reflect my readiness to continue growing toward a career in software development.
